# dotnet-api-demo

A simple .NET project demo with REST API, GitHub, and GitHub Actions.

## Tests

Tests are using xunit.

```bash
dotnet test
```

## Dependencies

```bash
dotnet restore
```

## Build

```bash
dotnet build
```

## Docker build

```bash
docker build -t dotnet-demo-api .
```
